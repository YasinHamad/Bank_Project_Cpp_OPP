# Bank_Project_Cpp_OOP
This project is related to course number 11 at [ProgrammingAdvices](https://programmingadvices.com/).   

The purpose of this project is :  
- Practice programming fundamentals.  
- Practice OOP.  

This is the OOP version of -with some additional features- this project [Bank_Project_Cpp](https://github.com/YasinHamad/Bank_Project_Cpp) where I built everything only as functions.

# Teck Stack
- C++  
- Files (to store data)  

# Features
- CRUD operations for clients.
- CRUD operations for system users.
- You can specify the features the user can access in the system (Authorization).
- Login feature (Authentication).
- Password encryption.
- Transactions
	- Withdraw.
	- Deposit.
	- Transfer money between accounts.
- Logs
	- Display all transfer operations performed in the system.
	- Display all logins to the system.
- Currency Exchange
	- List all currencies.
	- Search for a currency (by country name/code).
	- Update currency rate.
	- Currency calculator.
